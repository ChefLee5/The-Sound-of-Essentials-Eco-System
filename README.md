# The Sound of Essentials: Rhythm Quest - Web

This is the official landing page for **The Sound of Essentials: Rhythm Quest**.

## Features
- **Premium Design**: Gold and Dark Deep Space theme.
- **Story Integration**: Introduces Chapter 1 and the characters.
- **Asset Showcase**: Displays character art and cover images.
- **Music Preview**: Lists available tracks (functionality can be added).

## Setup
1. Navigate to this folder: `cd web`
2. Install dependencies (if not already): `npm install`
3. Run dev server: `npm run dev`

## Tech Stack
- React
- Vite
- Vanilla CSS with CSS Variables (Premium Theme)
