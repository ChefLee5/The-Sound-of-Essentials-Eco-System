# 🎶 The Sound of Essentials: Rhythm Quest

**Official Ecosystem Repository**

---

## What This Is

The Sound of Essentials: Rhythm Quest is a music-education adventure game and interactive experience built for children. Players explore the SOE universe through rhythm challenges, character-driven storytelling, and musical discovery — learning fundamental music concepts through play.

This repository contains the official web landing page, production scripts, and the full ecosystem scaffold for the Rhythm Quest experience.

## Getting Started

```bash
npm install
npm run dev
```

The dev server will start at `http://localhost:5173` by default.

### Build for Production

```bash
npm run build
npm run preview
```
